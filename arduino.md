## Arduino Robotics & Autonomous Systems Project (May 2025 - August 2025) 
Personal Summer Project   

### Objective
Developed an autonomous robotic platform to gain hands‑on experience in embedded systems engineering, C++ programming, sensing technologies, and electromechanical integration. The project progressed from a basic line‑following robot to an enhanced autonomous system capable of obstacle detection, navigation, and debris collection.

### Engineering Tools and Methods  
- **Programming & Embedded Systems:** Embedded C++, Arduino framework, microcontroller programming, real‑time control logic  
- **Robotics & Automation:** Autonomous navigation, line‑following control, obstacle detection and avoidance  
- **Sensors & Hardware:** Optical and infrared sensors, DC motors and motor drivers, hardware‑software integration  
- **Electrical & Mechanical Engineering:** Electrical wiring, signal troubleshooting, mechanical assembly, rapid prototyping  
- **Engineering Practices:** Sensor calibration, system integration and testing, root‑cause analysis, iterative design improvement  

### Initial Line Following Robot  
- Built an Arduino‑based mobile robot from a commercial kit, assembling mechanical, electrical, and embedded software subsystems.
- Developed sensor‑based control algorithms in C++ to enable accurate line‑following behavior using optical sensors.
- Integrated and validated:
  - Optical / line‑tracking sensors
  - DC motors and motor driver circuits
  - Microcontroller‑based control and power architecture

### Autonomous Vacuum System Modification  
- Extended functionality beyond the original kit design to create a fully autonomous mobile robot.
- Designed and implemented additional hardware including:
  - Infrared proximity sensors for real‑time obstacle detection and avoidance
  - A custom vacuum mechanism using a repurposed computer cooling fan, mechanically and electrically integrated into the robot chassis
- Programmed C++‑based autonomous navigation logic, enabling the robot to:
  - Detect obstacles dynamically
  - Execute collision‑avoidance maneuvers
  - Navigate enclosed environments without human input

### Results
- Developed a fully autonomous robotic platform progressing from a basic line‑following system to a mobile robot capable of obstacle avoidance and debris collection.
- Achieved reliable sensor‑based control, maintaining stable line tracking under varying lighting conditions and enabling collision‑free navigation in enclosed environments using infrared sensing.
- Implemented autonomous navigation logic to detect obstacles in real time and dynamically adjust robot path without human input.
- Designed and integrated a custom vacuum mechanism alongside sensing and drive systems, ensuring continuous operation during autonomous movement.
- Improved mechanical and electrical integration to achieve robust system performance, consistent motor control, and reliable sensor feedback.
- Validated embedded control design, sensor integration, and rapid mechanical iteration through successful development of a complete electromechanical prototype.
